# GBA-TEST-

儿童注意力与冲动控制游戏化筛查 V1.0。

本项目是一个可直接运行的静态网页原型，用于演示并试测两个游戏化测评任务：

1. **森林寻宝**：测量选择性注意、持续注意、抗干扰能力。
2. **太空刹车**：测量冲动控制、反应抑制能力。

## 运行方式

直接用浏览器打开 `index.html` 即可运行，不需要安装依赖。

## 当前版本功能

- 基本信息填写
- 儿童版指导语
- 森林寻宝练习与正式任务
- 太空刹车练习与正式任务
- 自动记录 trial 级原始行为数据
- 自动计算正确率、漏选率、误选率、反应时、反应时波动、持续注意下降值等指标
- 自动生成低/中/高风险结果
- 自动生成简版报告
- 支持导出 CSV 原始数据
- 支持导出 JSON 完整测评结果

## 重要说明

本项目仅为游戏化筛查原型，不作为临床诊断工具。报告中只提供风险线索提示，不能直接判断儿童是否存在 ADHD 或其他障碍。
