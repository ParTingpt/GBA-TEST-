# 儿童注意力与冲动控制游戏化筛查需求说明 V1.0

## 1. 产品定位

本项目是一套面向 6–12 岁儿童的游戏化筛查原型，通过儿童在游戏任务中的点击反应、正确率、错误率、漏选情况、反应时间等行为数据，对选择性注意、持续注意、抗干扰能力和冲动控制能力进行初步筛查。

本项目不作为临床诊断工具，不直接判断 ADHD 或其他障碍，仅用于提示是否存在进一步评估的必要。

## 2. 测评任务

### 2.1 森林寻宝

测量能力：

- 选择性注意
- 持续注意
- 抗干扰能力
- 视觉搜索速度
- 注意稳定性

任务规则：

屏幕中随机出现动物刺激，儿童只点击目标动物“小狐狸”，看到其他动物时不点击。

核心记录字段：

- target_total
- distractor_total
- hit_count
- miss_count
- false_alarm_count
- correct_rejection_count
- mean_reaction_time
- reaction_time_sd
- first_half_accuracy
- second_half_accuracy
- attention_decline

### 2.2 太空刹车

测量能力：

- 冲动控制
- 反应抑制
- 规则保持
- 错误控制
- 反应速度

任务规则：

看到绿色飞船点击，看到红色飞船不点击。

核心记录字段：

- go_total
- nogo_total
- go_hit_count
- go_miss_count
- nogo_correct_count
- nogo_error_count
- mean_go_reaction_time
- reaction_time_sd
- inhibition_error_rate

## 3. 风险等级规则

### 森林寻宝

- 低风险：正确率 ≥ 80%，误选率 ≤ 15%，持续注意下降值 < 20%。
- 中风险：正确率 60%–79%，或误选率 16%–30%，或持续注意下降值 ≥ 20%。
- 高风险：正确率 < 60%，或误选率 > 30%。

### 太空刹车

- 低风险：No-Go 错误率 ≤ 15%，Go 正确率 ≥ 80%。
- 中风险：No-Go 错误率 16%–30%，或 Go 正确率 60%–79%。
- 高风险：No-Go 错误率 > 30%，或 Go 正确率 < 60%。

### 总体风险

- 两个游戏均低风险：总体低风险。
- 任意一个游戏中风险，且没有高风险：总体中风险。
- 任意一个游戏高风险：总体高风险。

## 4. 第一版功能范围

- 基本信息填写
- 儿童版指导语
- 森林寻宝练习与正式任务
- 太空刹车练习与正式任务
- 原始行为数据记录
- 自动计算指标
- 自动生成低/中/高风险
- 简版报告
- CSV 导出
- JSON 导出
